# Restaurant Page - Trattoria Luna Rossa

A dynamic single-page restaurant website built with vanilla JavaScript and Webpack, created as part of [The Odin Project](https://www.theodinproject.com/lessons/node-path-javascript-restaurant-page) curriculum.

## Live Demo

https://niwaCS.github.io/restaurant-project

## About

Trattoria Luna Rossa is a fictional Italian restaurant website that demonstrates DOM manipulation using JavaScript. The entire page content is generated dynamically through JavaScript modules, with tabbed navigation allowing users to switch between Home, Menu, and About pages without reloading.

## Features

- **Dynamic Content Rendering**: All page content is created and managed through JavaScript
- **Tabbed Navigation**: Seamless switching between Home, Menu, and About sections
- **Modular Architecture**: Each page section is contained in its own ES6 module
- **Reservation Modal**: Interactive reservation form with validation
- **Responsive Design**: Clean, readable layout across devices
- **Hero Section**: Background image with photo credit attribution

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6 Modules)
- Webpack 5
- npm
